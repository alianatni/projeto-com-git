# Documentação do Projeto: Cadastro de Animais com Python


## Processo de Desenvolvimento
- Objetivo: Desenvolver um sistema para cadastro de animais (cachorros e gatos).
- Planejamento: A estrutura foi planejada com classes separadas para cada tipo de animal e uma interface simples.
- Desafios: Tive dificuldade em gerenciar as interações entre as classes e a interface e em organizar o código em cada arquivo.


## Uso do Git
- Criar branches para cada funcionalidade específica do código.
- Realizar commits frequentes para garantir um histórico correto das mudanças.
