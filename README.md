# projeto-com-git


# Objetivo
O objetivo deste projeto é criar um sistema simples em Python que permita cadastrar e exibir informações de animais (Cachorros e Gatos) utilizando uma interface gráfica construída com tkinter.


# Funcionalidades
1. Cadastrar animais com:
   - Nome
   - Idade
   - Porte (no caso de Cachorros)
   - Raça (no caso de Gatos)
2. Exibir uma lista com os dados de todos os animais cadastrados.


# Cronograma
Dia 1 = Configuração do repositório e estrutura inicial do projeto.
Dia 2 = Implementação da classe abstrata `Animal`.
Dia 3 = Desenvolvimento das classes derivadas: `Cachorro` e `Gato`.
Dia 4 = Construção da interface gráfica com tkinter.
Dia 5 = Testes, melhorias e documentação
